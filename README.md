#Sealion

自分のTODOを管理するためのアプリです。sealion = トド。そういうことです。  
色々タスク管理アプリ使ってみましたが、結局長続きしませんでした。  
と、ツールのせいにしてるわけですが、自分で作ればそんなこともできなくなります。

目指すところは、TaskChute的に流れるようにタスクを処理できるようなものになります。  
理想的には、システムが次にやることを指示してくれるようになることです。
